\section{Data}

The data collected came from CasA and VirgoA, with a potential hydrogen emission found in CasA. 
Each file includes the frequency and amp levels for trace 1 and trace 2.
Trace 1 is the data collected from when the radio telescope was pointed at the observation point.
Trace 2 is the data collected from when the radio telescope was pointed slightly away from the observation point.
If hydrogen was detected, then there would be a spike in trace 1, and no spike in trace 2.
